## Tips for Good Screenshots

1. **Use browser DevTools** to set a consistent viewport (1920x1080)
2. **Hide browser chrome** - use full page screenshots
3. **Use real-looking data** - not "test" or "lorem ipsum"
4. **Dark mode optional** - but be consistent
5. **Compress images** - use TinyPNG or similar to reduce file size

## How to Take Screenshots

### Option 1: Browser DevTools

1. Open DevTools (F12)
2. Click device toolbar icon
3. Set to "Responsive" with 1920x1080
4. Right-click > "Capture screenshot"

### Option 2: Dedicated Tool

- macOS: CleanShot X, or Cmd+Shift+4
- Windows: Snipping Tool, or Win+Shift+S
- Browser: Full Page Screen Capture extension

### Option 3: If you don't have access to the project

- Create mockups using Figma
- Use screenshots from your development environment
- Ask teammates for screenshots (for company projects)
